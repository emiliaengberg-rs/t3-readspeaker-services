=============================================================
ReadSpeaker Services
=============================================================

:Version:
   |release|

:Language:
   en

:Authors:
   Andreas Stenberg

:Email:
   andreas.stenberg@readspeaker.com

:License:

   `CC BY-NC-ND 4.0 <https://creativecommons.org/licenses/by-nc-nd/4.0/>`__ (Creative Commons)
   license

ReadSpeaker Services is an extension that facilitates the integration of ReadSpeaker webReader and ReadSpeaker docReader on your TYPO3 website.

.. note::

   You need a valid license for any of the ReadSpeaker products you are going to use.
   Visit `www.readspeaker.com <https://www.readspeaker.com>`_ to learn more about our products.

.. warning::

   Do not install this extension unless you have a ReadSpeaker license, it will not work without one.
